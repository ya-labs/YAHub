# Fluxo de trabalho do YA Hub

O YA Hub segue o fluxo oficial da YA LABS documentado no Handbook:

- [Fluxo de trabalho com GitHub](https://github.com/ya-labs/Handbook/blob/main/docs/processos/fluxo-de-trabalho-github.md)

Use o Handbook como fonte principal para:

- issues;
- GitHub Projects;
- branches;
- commits;
- Pull Requests;
- releases;
- validações.

## Adaptações locais

No momento, o YA Hub não possui exceções específicas ao fluxo da YA LABS.

Se o projeto precisar de alguma regra própria no futuro, registre apenas a diferença neste arquivo e mantenha o processo geral no Handbook.
