# Assets da hero

Os arquivos desta pasta recompõem a direção visual aprovada da hero dark premium.

## Camadas

Use as camadas nesta ordem, do fundo para a frente:

1. `fundo-base.svg`
2. `grid-tecnico.svg`, alinhado à direita e com baixa opacidade
3. `montanha.png`, alinhada à direita
4. `orbita.svg`, sobre a montanha
5. `particulas.svg`
6. `estrela-principal.svg` e `estrela-pequena.svg`
7. `marca-horizontal.svg`

## Marca

- `simbolo-ya.svg`: símbolo robusto claro, baseado na referência oficial;
- `simbolo-ya-escuro.svg`: símbolo institucional para fundos claros;
- `simbolo-ya-monocromatico.svg`: versão de uma cor usando `currentColor`;
- `simbolo-ya-fino.svg`: versão fina usada na aplicação vertical;
- `simbolo-ya-fino-escuro.svg`: versão fina para fundos claros;
- `marca-horizontal.svg`: assinatura horizontal clara;
- `marca-horizontal-fina.svg`: assinatura horizontal fina usada na hero;
- `marca-horizontal-escura.svg`: assinatura horizontal para fundos claros;
- `marca-vertical.svg`: assinatura vertical;
- `estrela-marca.svg`: estrela dourada isolada;
- `favicon-ya.svg`: ícone compacto para navegador e atalhos.

`referencia-logo-oficial.png` e `referencia-simbolo-oficial.png` preservam os materiais enviados pelo responsável da marca. `referencia-hero.png` preserva o mockup original usado como régua visual.

## Observação

O mockup original é uma imagem raster achatada. Por isso, a montanha foi reconstruída e isolada com transparência, enquanto os elementos geométricos foram refeitos como SVG editável para evitar recortes com resíduos de outras camadas.
